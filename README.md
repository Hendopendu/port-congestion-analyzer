# 🚢 Port Congestion & Freight Rate Analyzer

A web scraping project that analyzes the correlation between global port congestion and shipping freight rates.

## 📊 Project Overview

This project collects data from various shipping and port sources to identify relationships between port congestion levels and container freight rates. The analysis helps understand supply chain bottlenecks and their impact on shipping costs.

## 🛠️ Tech Stack

- **Python** (Web Scraping & Data Analysis)
- **Docker** (Containerization)
- **Beautiful Soup** & **Requests** (Web Scraping)
- **Pandas** (Data Analysis)
